Binary Tree Visualizer

A web-based binary tree visualization tool developed using HTML, CSS, and JavaScript. The application allows users to create, insert, and visualize binary tree structures interactively while demonstrating how nodes are connected and traversed in real time.
HTML CSS JavaScript Data Structures Visualization
  • Interactive binary tree node insertion and visualization
  • Real-time rendering of parent and child node connections
  • Supports tree traversal demonstrations such as Inorder, Preorder, and Postorder
  • Dynamic tree updates using JavaScript DOM manipulation
  • User-friendly interface for learning binary tree structures
One of the main challenges was dynamically positioning tree nodes and maintaining proper spacing as the tree structure expanded. Implementing traversal visualizations while keeping the animations synchronized with the tree logic also required careful JavaScript handling.
The project successfully demonstrated binary tree concepts through interactive visualization and real-time traversal animations. It serves as an educational tool for understanding tree structures, node relationships, and traversal algorithms using web technologies.

OS Preemptive Scheduler Calculator

A Python-based operating system scheduler calculator that simulates and analyzes preemptive CPU scheduling algorithms. The application computes scheduling results and visually animates process execution, allowing users to better understand how different algorithms manage CPU time.
Python OS Scheduling Algorithm Simulation Visualization
  • Supports preemptive scheduling algorithms: SRTF, Priority, and Round Robin
  • Automatic calculation of waiting time, turnaround time, and completion time
  • Animated visualization of process execution timeline
  • Play and pause controls to step through scheduling results
  • User-friendly input for defining processes and scheduling parameters
Designing accurate preemptive scheduling logic required careful handling of context switching, time slicing, and priority changes. Translating calculated results into a clear and intuitive animation was also challenging, especially for maintaining synchronization between logic and visualization.
The project successfully demonstrated key operating system scheduling concepts through both computation and visualization. It serves as an effective educational tool for understanding how different preemptive algorithms affect CPU performance and process behavior.
